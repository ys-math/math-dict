# math-dict

数学のための日本語 IME 辞書です。

[English README](README.md)

数学でよく使用する単語や人名、記号などを含んだ日本語　IME　用の辞書です。

## 導入方法

[`dist/`](dist/) から使用している IME 用のファイルをダウンロードしてください。

| IME | ファイル | 取り込み方 |
|---|---|---|
| **Google 日本語入力 / Mozc** | [`dist/google/math_dict_ja.txt`](dist/google/math_dict_ja.txt) | 辞書ツール → 管理 → 新規辞書にインポート |
| **macOS 日本語入力** | [`dist/macos/math_dict_ja.plist`](dist/macos/math_dict_ja.plist) | システム設定 → キーボード → ユーザ辞書 にドラッグ＆ドロップ |
| **Microsoft IME** | [`dist/msime/math_dict_ja.txt`](dist/msime/math_dict_ja.txt) | 単語の登録 → ユーザー辞書ツール → ツール → テキストファイルからの登録 |
| **ATOK** | [`dist/atok/math_dict_ja.txt`](dist/atok/math_dict_ja.txt) | 辞書ユーティリティ → ツール → ファイルから登録・削除 |

収録語数は **1,076 語**です。論理・集合論、代数学、位相空間論、解析学、圏論、可換環論、代数幾何など 23 分野の用語のほか、数学記号 97 語、数学者名 68 名、ラテン文字表記の用語 24 語を収めています。数学者名はカタカナとラテン文字の両方で登録してあり、例えば`がろあ` からは ガロア と Galois が出ます。

> [!NOTE]
> 実際に動作確認したのは Google 日本語入力のみです。残る3形式は文字コード・BOM・改行コード・品詞名・plist の構造を検査しただけにとどまります。取り込みに失敗した場合は[Issue](../../issues) を立ててください。

## 使用方法

**正しい読み**に対して用語を割り当てています。略語による入力は対応していません。例えば**数学的帰納法**を入力する場合はその読みである `すうがくてききのうほう` を入力します。


## 語の追加方法

[`terms/`](terms/) の該当ファイルを編集してください。ここが唯一の原本で、`dist/` 以下は生成物です。

```
読み <TAB> 単語 <TAB> 品詞 <TAB> フラグ（任意）
じゅんどうけい	準同型	noun
```

品詞は IME に依存しない値を使います（`noun`、`noun-suru`（する接続）、`noun-adj`（な接続）、`person`、`symbol`、`latin`）。各 IME の品詞体系への対応づけはビルド側で行います。分野はファイル名から決まるため、行ごとに書く必要はありません。フラグは任意で、`common`（読みが日常語と衝突する）と `low-confidence`（読みに確認が必要）をカンマ区切りで指定できます。

編集したら次を実行してください。

```sh
python3 validate.py   # 形式を検査し、REVIEW.md を再生成する
python3 build.py      # terms/ から dist/ を生成する
```

どちらも CI で実行されます。CI はさらに、コミットされている `dist/` が `terms/` から生成される内容と一致するかをチェックします。

`validate.py` は「読みのモーラ数は単語の漢字数以上」という規則で略語を弾きますが、これは下限にすぎません。`きのう→数学的帰納法`（3 モーラ / 6 漢字）は検出できても、`こゆう→固有値`（3 モーラ / 3 漢字）は通過します。CI ではなく差分を確認してください。

日常語と衝突する語、および読みが未確認の語は [REVIEW.md](REVIEW.md) にまとめてあります。

## 注意事項

この辞書の語は言語モデルが原案を作成し、機械的な検査を通したものです。検査で見つかるのは形式的な誤り（不正な仮名、重複、カタカナ語と読みの不一致）だけで、**形式は正しいのに実在しない語**は見つけられません。

読みの誤りはその項目が変換候補に出てこないだけですが**実在しない語は変換され、文章に紛れ込みます**。おかしな語を見かけたら [Issue](../../issues) を立てるか、該当行を削除してください。
